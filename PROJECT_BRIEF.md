## 1.1 세계관 (Lore)

주인공은 **하급 뱀파이어**. 뱀파이어 귀족 사회에서 지위를 올리는 유일한 방법은 **피의 도박** — 자신의 피(HP)를 걸고 상위 혈족에게 도전하는 포커 대결이다.

- **"Bloody"** — 피를 걸고 싸우는 도박, 뱀파이어의 본질
- **"Royal"** — 포커의 Royal Flush + 뱀파이어 귀족 왕좌
- **HP = 피(Blood)** — 베팅은 문자 그대로 자신의 피를 거는 행위
- **알파 카드** — 뱀파이어 혈족 고유 능력 (치트 스킬)
- **보스 전리품** — 상위 혈족을 쓰러뜨려 얻는 고유 권능
- **부활(판 뒤엎기)** — 뱀파이어의 불사 본능
- **카드 계승** — 패배한 자의 핏줄 흡수
- **스테이지 여정** — 숲(변방) → 성(귀족 영지) → 던전(왕의 지하) → 하늘(최종 왕좌)

## 3. 주제 (Theme)
- **피의 도박**: 매 라운드마다 자신의 피(HP)를 걸고 "공격적으로 밀어붙일지, 버티며 다음 판을 노릴지" 선택하는 리스크 관리. 뱀파이어에게 피는 곧 생명이자 화폐.
- **금기를 깨는 포커**: 일반 포커 규칙을 혈족 능력(알파 카드)으로 비틀어 역전의 서사를 만든다.
- **왕좌를 향한 등반**: 1~50 스테이지에서 점점 강해지는 상위 혈족을 카드/아이템 빌드와 판단으로 돌파하며 왕좌에 오른다.

## 4. 장르 (Genre)
- **메인 장르**: 덱빌딩 로그라이크 포커 RPG
- **서브 장르**: 턴제 전략, 하이리스크 하이리턴, 보스러시형 스테이지 클리어
- **포지셔닝**: "덱빌딩 로그라이크 포커게임 + 치트 스킬 배틀"

## 5. 타깃 유저 (Target Audience)
- **코어**: 인디/로그라이크 유저, 카드게임(포커·덱빌딩) 선호층, 리스크-리워드 전략 플레이어
- **확장**: 짧은 세션 긴장감을 원하는 캐주얼 유저, 스트리밍 시청형(극적 역전) 콘텐츠 소비자

## 6. 핵심 재미와 제공 경험

### 6.1 생존 긴장감
- HP가 곧 자원이라, 베팅 한 번이 생존 여부를 가른다.
- `FOLD / CALL / RAISE / ALL-IN` 선택이 즉시 체력 손익으로 체감된다.

### 6.2 치트 기반 역전 쾌감
- 천리안, 재장전, 운명 교환, 단두대 등 알파 카드로 불리한 패를 뒤집는다.
- 보스 전리품 카드는 "규칙 파괴" 수준의 역전 순간을 만든다.

### 6.3 빌드 메이킹의 재미
- 스테이지 클리어 후 카드 보상 선택, 중복 카드 레벨업, 상점 아이템 구매로 플레이 스타일 분기.
- 스테이지마다 장착 카드/아이템을 재구성하는 준비 단계가 전략 깊이를 만든다.

### 6.4 구간별 공략 경험
- 스테이지 구간마다 적 AI 성향이 달라 공략법이 바뀐다 (→ 7.4절 참조).
- 10단위 보스와 50층 최종전이 중간·최종 목표를 분명하게 제공한다.

### 6.5 실패-재도전 루프
- 패배 시 **1장의 카드를 계승**해 다음 런의 기반을 만든다.
- **"판 뒤엎기" 부활** (런당 3회) — 실패를 즉시 재도전의 드라마로 전환한다.

## 7. 게임 시스템 상세

### 7.1 게임 루프 (Phase Flow)

```
MENU → CARD_REWARD → [SHOP (5스테이지 주기)] → DECKBUILDING → DISCARD
  → PREFLOP → FLOP → TURN → RIVER → SHOWDOWN
  → 승리: VICTORY → 다음 스테이지
  → 패배: REVIVE_PROMPT (부활 잔여 시) / GAMEOVER → SCORE_PROMPT → SCOREBOARD
```

### 7.2 HP·베팅 밸런스

| 상수 | 값 | 설명 |
|------|-----|------|
| 초기 HP | 100 | `INITIAL_HP` |
| 최대 HP 한도 | 999 | `MAX_PLAYER_HP_LIMIT` |
| Ante (참가비) | 5 HP/스테이지 | `HP_PER_STAGE` |
| 적 HP 공식 | 구간별 선형 증가 | 1~10: 46~100, 11~20: 168~240, 21~30: 310~400, 31~40: 492~600, 41~50: 724~850 |
| 보스 HP | 기본 HP × 1.5 | 10/20/30/40/50 스테이지 |

#### 사용 규칙
- **스테이지당 충전 횟수**: 카드별 1~3회 (레벨에 따라 증가)
- **라운드 쿨다운**: 카드별 1~3라운드
- **공격 카드 인텔**: 35% 확률로 상대 패 1장 추가 공개
- **배타 그룹**: 천리안(`PEEK_OPPONENT`)과 마안(`BOSS_EYE`)은 VISION 그룹으로 중복 장착 불가 (상위 카드만 유지)

### 7.4 적 AI 시스템 (5종 성향)

| 구간 | AI 성향 | 등장 혈족 | 특징 |
|------|---------|----------|------|
| 1~9 | 겁쟁이 (Coward) | 혈귀 → 종자 → 야수 → 혈기사 | 큰 배팅에 쉽게 폴드, 블러핑 효과적 |
| **10** | **보스** | **남작 (Baron)** | **첫 번째 귀족, 위압적** |
| 11~19 | 다혈질 (Hothead) | 사냥꾼 → 광신도 | 웬만하면 콜, 블러핑 빈번, 기본 배팅 +10 |
| **20** | **보스** | **자작 (Viscount)** | **탐욕스러운 귀족** |
| 21~29 | 신중 (Cautious) | 원로 | 강한 패 아니면 소액에도 폴드, 레이즈 거의 없음 |
| **30** | **보스** | **백작 (Count)** | **냉혹한 처형자** |
| 31~39 | 합리 (Rational) | 근위대장 | 기대값 기반, HP 비율 고려 폴드 |
| **40** | **보스** | **공작 (Duke)** | **혈왕 직하 최상위 귀족** |
| 41~49 | 보스급 (Boss) | 혈위병 | 15% 이하 폴드율, 대형 레이즈 |
| **50** | **최종 보스** | **혈왕 (Blood King)** | **만 년의 왕좌, 최종 결전** |

**공통 AI 메커니즘**:
- 핸드 강도 4단계 평가 (약(-1) / 보통(0) / 강(1) / 매우강(2))
- 10% 블러핑 확률, 5% 실수(미스플레이) 확률
- 배팅 상한: `(40 + 스테이지 × 2) × 난이도배수`
- 난이도 배수(`difficultyMultiplier`)에 의한 전체 스케일링

## 9. 차별화 포인트
- "포커 + 로그라이크" 결합에 그치지 않고, **치트 카드/보스 카드로 메타 규칙 자체를 흔드는 설계**.
- HP를 화폐/체력/공격 자원으로 통합해, **모든 액션이 명확한 위험과 보상**을 갖는다.
- 카드 성장(레벨), 아이템, 스테이지 AI 성향이 결합된 **높은 반복 플레이 가치**.
- 5종 AI 성향에 의한 **구간별 전략 전환** — 같은 포커 룰 위에서 공략이 바뀐다.
